# Tic Tac Toe Game (AI & Friend Mode)

A responsive Tic-Tac-Toe web application built using React and Vite.

## Features
- Play with AI (Minimax Algorithm)
- Play with Friend (Local Multiplayer)
- Modern UI with animations
- Winning line highlight
- Responsive design

## Tech Stack
- React.js
- JavaScript
- Vite
- CSS

## Game Modes
1. Human vs AI
2. Human vs Human

## How to Run Locally

```bash
npm install
npm run dev
